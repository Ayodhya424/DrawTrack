# DrawTrack
DrawTrack is an AI-powered virtual whiteboard that lets users draw in the air using hand tracking. Built with OpenCV and MediaPipe, it captures real-time gestures from a webcam and translates them into strokes on a digital canvas. 

# Note: 
This project is currently on hold while pen detection using machine learning is under process. Future updates will include pen tip tracking and shape auto-correction. Anyone who wants to contribute can contact me through mail.
